# Currency Converter

A simple web app that converts currencies using a live API.

## Features
- Convert between multiple currencies  
- Live exchange rates  
- Flag updates based on country  
- Dynamic result update  

## Tech Used
- HTML  
- CSS  
- JavaScript  
- Currency API (fawazahmed0)
